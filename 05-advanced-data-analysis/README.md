# Análisis Avanzado de Datos

Esta carpeta contiene proyectos de análisis de datos avanzados que cubren diferentes tipos de problemas de machine learning y análisis de datos, desde clasificación y regresión hasta análisis de datos climáticos.

## Estructura de Carpetas

```
05-advanced-data-analysis/
├── README.md                    # Este archivo
├── boston-housing/              # Análisis de precios de viviendas
├── iris-species/                # Análisis de especies de Iris
└── global-climate/              # Análisis de datos climáticos globales
```

## Proyectos Incluidos

---

### 1. Iris Species Analysis

**Ubicación**: `iris-species/`

Análisis de tres especies de flores Iris (setosa, versicolor, virginica) basándose en las medidas de sépalos y pétalos.

**Ver más detalles**: [README de Iris Species](iris-species/README.md)

### 1. Boston Housing Price Analysis

**Ubicación**: `boston-housing/`

Análisis del valor mediano de viviendas en el área metropolitana de Boston basándose en características socioeconómicas y ambientales del área.

**Ver más detalles**: [README de Boston Housing](boston-housing/README.md)

---

### 3. Global Climate Data Analysis

**Ubicación**: `global-climate/`

Análisis climatológico de temperatura utilizando datos ERA5 (reanálisis climático) para visualizar patrones de temperatura global.

**Ver más detalles**: [README de Global Climate](global-climate/README.md)

## Requisitos

Los proyectos utilizan principalmente:

- **Python** con librerías estándar de data science:
  - `pandas` - Manipulación de datos
  - `numpy` - Operaciones numéricas
  - `xarray` - Manejo de datos en 3 dimensiones
  - `matplotlib` / `seaborn` - Visualización
  - `scikit-learn` - Machine learning
  - `jupyter` - Notebooks interactivos

## Cómo Empezar

1. Navega a la carpeta del proyecto que te interese
2. Lee el README.md específico del proyecto para más detalles
3. Abre el notebook Jupyter correspondiente
4. Sigue las instrucciones y ejercicios en el notebook

## Notas

- Cada proyecto es independiente y puede realizarse por separado
- Los datasets están incluidos en las carpetas `data/` de cada proyecto
- Los notebooks contienen código ejecutable y explicaciones paso a paso
